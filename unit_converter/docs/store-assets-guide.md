# Store Assets Guide

## Screenshot Requirements

### Current Screenshots
Located in: `android/fastlane/metadata/android/en-US/images/`

### Recommended Screenshot Order (Phone - 4 screenshots)

**Screenshot 1: Hero/App Intro (MANDATORY)**
- Full-screen phone mockup showing the main conversion screen
- App name prominently displayed: "Unit Converter Pro"
- Highlight: Custom Units feature
- Show a custom unit being used (e.g., "Furlong" conversion)
- Text overlay: "Only app with Custom Units"

**Screenshot 2: Currency Feature**
- Currency converter screen
- Show 150+ currencies indicator
- Display live rate conversion (USD to EUR)
- Clean, professional look

**Screenshot 3: Category Overview**
- Show all 10 categories in a grid or list
- Highlight the uniqueness (categories competitors have + custom)
- Show the clean UI and theme

**Screenshot 4: Quick Presets & Favorites**
- Show the preset chips (°F↔°C, kg↔lb, etc.)
- Show favorites or recent history
- Demonstrate everyday usability

### Promo Video Requirements (30 seconds)

**Concept:** "The Only Unit Converter With YOUR Units"

**Storyboard:**
0-5s: Open on generic converter app showing same 5 units. User looks frustrated.
5-10s: Transition to Unit Converter Pro. Highlight "Add Custom Unit" button.
10-20s: Create a custom unit (e.g., "Furlong" for horse racing fan). Show conversion works.
20-25s: Show 150+ currencies with live rates.
25-30s: Close with app icon and "Unit Converter Pro" title, "Custom Units" tagline.

**Technical Specs:**
- Format: MP4 or WebM
- Resolution: 1920x1080 (minimum)
- Duration: 15-30 seconds
- No audio required (or short original music)

### Feature Graphic (1200x628)
- Clean gradient background (use one of the 3 app themes)
- App icon prominently displayed
- Text: "Custom Units • 150+ Currencies • 10 Categories"
- Must be readable at small sizes

## Current Issues with Screenshots

Based on Play Store review:

1. **Generic appearance** - Screenshots look like every other converter app
2. **No hero image** - First screenshot doesn't communicate unique value
3. **No promo video** - Competitors have videos, you don't
4. **Text too small** - Descriptions in screenshots aren't legible at store size

## Quick Wins

1. **Reorder screenshots** to put Custom Units first
2. **Add text overlays** to highlight key features
3. **Create a simple promo video** using Screen Recorder + DaVinci Resolve (free)
4. **Update feature graphic** to be more professional
